A **Machine Learning based web application** that detects brain diseases from **MRI images**.
The system predicts diseases such as **Brain Tumor, Parkinson’s Disease, and Alzheimer’s Disease** using deep learning models.

---

## 🚀 Project Overview

Brain diseases are difficult to detect in early stages.
This project uses **Machine Learning and Deep Learning** to analyze MRI images and predict brain diseases automatically.

Users can upload MRI scans and receive **instant predictions** through a **Streamlit web application**.

---

## 🎯 Objectives

* Detect brain diseases from MRI images
* Classify multiple brain diseases
* Provide **real-time prediction**
* Improve early diagnosis using ML

Diseases detected:

* Brain Tumor
* Parkinson’s Disease
* Alzheimer’s Disease

---

## 🧠 Algorithms Used

* **CNN (Convolutional Neural Network)**
* **DenseNet-121**
* **LSTM (Long Short-Term Memory)**

These models analyze MRI images and classify the disease patterns.
